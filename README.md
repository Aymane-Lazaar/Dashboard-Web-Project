# Dashboard Web Project

This project is a simple **HTML and CSS dashboard** interface featuring a sidebar menu, data cards, and a product table. It uses Font Awesome for icons and includes hover effects and responsive design elements.

## Project Structure


- `index.html` → Main HTML file containing the dashboard structure.
- `style dashboard.css` → CSS file for styling the dashboard, sidebar, cards, and table.
- `photo/user-200x300.webp` → User profile image displayed in the sidebar.

## Features

- **Sidebar menu** with icons and expandable width on hover.
- **Profile section** with user image and name.
- **Data cards** displaying users, posts, products, and revenue with hover effects.
- **Products table** showing product name, price, and count.
- Responsive design for smaller screens.
- Smooth hover animations on buttons, cards, and table rows.
